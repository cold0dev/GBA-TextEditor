# GBA-TextEditor
Text editor for Gameboy Advance  
Build with DevkitPro  

# Usage
View Mode:  
Dpad Up - Move selector up  
Dpad Down - Move selector down  
B - Insert new line  
A - Edit selected line  
  
Edit Mode:  
Dpad Left - Move selector left  
Dpad Right - Move selector right  
B - Space  
A - Finish editing
